# cart

Carro de compras, realizado en vue 3

## Pasos

1 Asegurarse de tener node v18.12.1 (npm v8.19.2)

### Instalar

```sh
npm install
```

### Ejecutar

```sh
npm run dev -- --host --port=8080
```
